# 英文單字學習 App — 專案筆記

> 開始日期：2026-06-01
> 開發者：詹爸（Jack）+ 兒子 John（高一）
> 目標：父子一起學 AI，用 LIVE ABC 英語核心字彙做一個學習 App

---

## 目前產品方向（2026-06-03 更新）

- 主要使用者先聚焦 John。
- John 目前就讀上海台商子女學校，高一。
- 暫時不考慮 Ivy 的使用情境。
- 開發重點改為「高一學生」「學測準備」「可持續練習」。
- 這個 App 的定位是幫 John 平時學習、複習，並為高三參加台灣學測考試做長期準備。
- 之後功能設計不需要刻意降低到小學生操作難度。
- App 方向從單純英文單字工具，逐步擴充為 John 的學測五科複習工具。
- 首頁五科入口是架構隔離用：未來開發其他科目時，不能影響已經完成的科目。
- 各科開發時，程式功能與題庫資料必須盡量分開；未來新增題庫應優先只改題庫檔，不動主程式。

## 單字資料來源

- **課本**：LIVE ABC 英語核心字彙完全攻略
- **實際資料**：大考中心「高中英文參考詞彙表（111學年度）」Level 2-4
- **來源網址**：由 Threads 用戶 @copper0722 整理成 Google Sheets 公開分享
- **單字數量**：3,005 個（Level 2：1002字、Level 3：1001字、Level 4：1002字）
- **欄位**：級別、英文單字、詞性、中文翻譯

## 已完成功能（2026-06-01）

### 單字卡模式
- [x] 3D 翻轉動畫：正面英文＋詞性，翻面顯示中文
- [x] 🔊 真人美式發音（優先 FreeDictionary API，找不到備援瀏覽器 TTS）
- [x] ✓ 已學 / 🔴 困難 標記，再點一次取消
- [x] 隨機排序 / 順序排列
- [x] 鍵盤操作：← → 切換、空白/Enter 翻面、`k` 已學、`h` 困難

### 級別篩選
- [x] 全部 / Level 2 / Level 3 / Level 4 / 困難單字
- [x] 今日學習模式（困難字優先 + 補足新字到 10 個）

### 測驗模式
- [x] **選擇題**：看英文選中文（4 選 1）
- [x] **拼字測驗**：看中文打英文（鍵盤輸入，Enter 確認）
- [x] 答對顯示熟練度：⚡快速（≤3秒）、👍普通（≤7秒）、📚需加強（>7秒）
- [x] 💡 提示按鈕（顯示首字母＋字數）
- [x] 每次隨機抽 20 題

### 計時功能
- [x] 每題 12 秒倒數計時條（彩色）
- [x] 時間到自動算錯、標為困難
- [x] 記錄答題反應時間，影響熟練度評分

### 每日學習目標
- [x] 頂部綠色進度條，目標每天 10 個新字標為「已學」
- [x] 每翻單字卡自動記錄今日學習
- [x] 測驗答題也計入今日紀錄
- [x] 「開始今日學習」按鈕：困難字優先 + 自動補新字

### 進度追蹤
- [x] 整體統計：已學 / 困難 / 未接觸 / 完成度 %
- [x] 各級別學習進度條
- [x] 困難單字清單（可點擊直接標為已學）
- [x] 重設進度功能

### 家長查看（詹爸專用）
- [x] 右上角「👨‍👦 家長查看」按鈕
- [x] 報告內容：整體完成度、今日學習統計、各級進度條、近期測驗成績、困難單字清單
- [x] 測驗成績自動儲存（最近 30 次）

### 資料儲存
- 全部存在瀏覽器 LocalStorage，不需要後端伺服器
- `vocab_progress`：每個單字的狀態（已學/困難）
- `vocab_daily`：每日學習記錄（含答題時間）
- `vocab_quiz_scores`：測驗成績歷史
- `vocab_log`：學習行為日誌（最近 500 筆）
